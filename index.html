FAQ

Real questions,
real answers.

Yes. The entire app — all 12 tools — is free and open source. You only pay for your own API key usage (e.g. Gemini's pay-as-you-go pricing, which is very cheap). If you use a local LLM like Ollama, the cost is literally zero.
Only to the AI provider you configure (e.g. Google for Gemini), and only through your own API key. Wrytica itself never sees, stores, or touches your text. If you use Ollama locally, no text leaves your machine at all.
Wrytica supports Google Gemini (gemini-2.0-flash and others), LM Studio (any GGUF model via local server), and Ollama (any model you have pulled). You can switch models per-session from Settings without restarting.
Upload PDFs, text files, or Word documents. Wrytica indexes them using PageIndex (semantic page-level chunking) combined with an optional vector cache. When you chat or run an agent, it retrieves the most relevant pages and injects them into the prompt context automatically.
No. Wrytica runs entirely in the browser — just open index.html from the downloaded zip. The optional Python backend unlocks deep PDF extraction (MinerU) and background OCR jobs, but the core app works without it.
PageIndex is Wrytica's built-in lightweight retrieval that ranks pages by keyword and semantic similarity — no external database needed. Hybrid mode adds an in-browser vector cache for faster lookups on large collections. Both are 100% local.
